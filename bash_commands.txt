ls|List directory contents. Example: ls -l
cd|Change directory. Example: cd /path/to/directory
mkdir|Create a new directory. Example: mkdir new_directory
cp|Copy files and directories. Example: cp file1 file2
mv|Move or rename files and directories. Example: mv file1 file2
rm|Remove files and directories. Example: rm file
grep|Search for patterns in files. Example: grep pattern file
sed|Stream editor for filtering and transforming text. Example: sed 's/old/new/g'
awk|Pattern scanning and text processing language. Example: awk '{print $1}' file
chmod|Change file permissions. Example: chmod 755 file
cat|Concatenate and display file content. Example: cat file
head|Output the first part of files. Example: head file
tail|Output the last part of files. Example: tail file
sort|Sort lines of text files. Example: sort file
uniq|Report or omit repeated lines. Example: uniq file
wc|Print newline, word, and byte counts for each file. Example: wc file
touch|Create an empty file or update the timestamp of an existing file. Example: touch filename
find|Search for files in a directory hierarchy. Example: find /path/to/directory -name "*.txt"
locate|Find files by name. Example: locate filename
tar|Archive files. Example: tar -czvf archive.tar.gz directory
zip|Compress files into a zip archive. Example: zip archive.zip file1 file2
unzip|Extract files from a zip archive. Example: unzip archive.zip
