ls|List directory contents. Usage: ls [OPTION]... [FILE]...
cd|Change directory. Usage: cd [DIRECTORY]
mkdir|Create a new directory. Usage: mkdir [OPTION]... DIRECTORY...
cp|Copy files and directories. Usage: cp [OPTION]... [-T] SOURCE DEST
mv|Move or rename files and directories. Usage: mv [OPTION]... [-T] SOURCE DEST
rm|Remove files or directories. Usage: rm [OPTION]... FILE...
grep|Search for patterns in files. Usage: grep [OPTION]... PATTERN [FILE]...
sed|Stream editor for filtering and transforming text. Usage: sed [OPTION]... {script-only-if-no-other-script} [input-file]...
awk|Pattern scanning and text processing language. Usage: awk [POSIX or GNU style options] -f program-file [--] [file(s)]
chmod|Change file permissions. Usage: chmod [OPTION]... MODE[,MODE]... FILE...
chown|Change file owner and group. Usage: chown [OPTION]... [OWNER][:[GROUP]] FILE...
ps|Report a snapshot of the current processes. Usage: ps [options]
kill|Send signals to processes or report process status. Usage: kill [options] <pid> [...]
df|Report file system disk space usage. Usage: df [OPTION]... [FILE]...
du|Estimate file space usage. Usage: du [OPTION]... [FILE]...
tar|Manipulate archive files. Usage: tar [OPTION...] [FILE]...
ssh|OpenSSH SSH client. Usage: ssh [user@]hostname [command]
scp|Secure copy (remote file copy program). Usage: scp [OPTION]... [-T] SOURCE DEST
grep|Search for patterns in files. Usage: grep [OPTION]... PATTERN [FILE]...
find|Search files and directories. Usage: find [OPTION] [PATH] [EXPRESSION]
history|Display or manipulate the history list. Usage: history [n]
